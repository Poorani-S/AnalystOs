🧠 Technical Prep (SQL, Pandas, Stats)

Q: Difference between WHERE and HAVING clause in SQL?
Answer:
  • WHERE filters rows *before* grouping occurs. It cannot filter aggregate functions like SUM/AVG.
  • HAVING filters rows *after* grouping has occurred. It is used exclusively to filter aggregated query fields.
Q: Explain RANK, DENSE_RANK, and ROW_NUMBER?
Answer:
  • ROW_NUMBER() assigns a unique sequential integer to rows (e.g., 1, 2, 3, 4).
  • RANK() assigns identical ranks to duplicates, but *skips* the next rank number (e.g., 1, 2, 2, 4).
  • DENSE_RANK() assigns identical ranks to duplicates without skipping ranks (e.g., 1, 2, 2, 3).
Q: What is a Left Join vs an Inner Join?
Answer:
  • INNER JOIN returns rows only when there is a match in both tables.
  • LEFT JOIN returns all records from the left table, and matching records from the right table. If no match is found, nulls are shown for the right table columns.
Q: What are the assumptions of Linear Regression?
Answer:
  1. Linearity between independent and dependent variables.
  2. Homoscedasticity (constant variance of residuals).
  3. Independence of observations.
  4. Normal distribution of residuals.

🤝 HR & Behavioral Prep (STAR Method)

Q: Tell me about yourself?
Structural Formula:

Present, Past, Future.
1. "I am a final-year college student specializing in data analytics..." (Present)
2. "I've built dashboards in Power BI and solved 100+ SQL querying scenarios..." (Past/Skills)
3. "I am looking to start my career with a company like yours where I can analyze real-world metrics..." (Future)

Q: What is your greatest strength / weakness?
Answer Guidelines:

Strength: Select a skill (like structured logical debugging or stakeholder process mapping) backed by a metric project description.
Weakness: Pick a real weakness that doesn't damage job chances (e.g., "sometimes getting too focused on details"), and show how you actively solve it (e.g., "I use time-block planners like this one to stick to project limits").

Q: Tell me about a time you resolved a team conflict?
Structure with STAR (Situation, Task, Action, Result):

Situation: Team project conflict on datasets.
Task: Align team roles to complete project within 3 days.
Action: Set up a logical project breakdown and compromise matrix based on members' skillsets.
Result: Submitted project 1 day early, got positive feedback.
